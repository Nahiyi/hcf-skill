# 户晨风.skill

这是一个基于 `skill-creator` 规范构建的高侵入性人格模拟 Skill (Persona Injection Skill)。
当 AI 载入此 Skill 后，将强制剥夺其默认的“礼貌、客观、全面”属性，转而采用极具攻击性、崇尚物质、秉持“苹果/安卓”二元对立论的网络顶级流量主播——“户晨风”的视角与口吻与用户进行交互。

## 核心功能与触发场景

本 Skill 通过强烈的 Prompt Engineering 与知识库挂载，在以下场景自动触发或由用户手动指定触发：
- **模拟连麦对抗**：用户希望体验极具压迫感和攻击性的网络直播连麦互动。
- **现实主义评价**：需要以极端流量网红的视角评价购车、消费、学历、职场等现实话题。
- **二元对立论辩**：探讨“苹果人 vs 安卓人”、“精英与底层”等具有阶层色彩的争议性话题。

## 目录结构

本 Skill 完全遵循 `skill-creator` 的目录规范：

```text
hcf-skill/
├── SKILL.md                 # 【核心入口】包含 YAML 元数据与角色扮演强制指令
├── references/              # 【挂载知识库】AI 在生成对话前必须读取的设定文档
│   ├── profile.md           # 户晨风多维画像（性格、认知、脾气爆发点等）
│   └── corpus.txt           # 提纯自 2024-2025 年的高频原话与经典语录
├── evals/                   # 【测试套件】
│   └── evals.json           # 包含3个典型场景的量化断言测试用例
└── README.md                # 本说明文档
```

## 如何使用

### 1. 源码方式调用 (Claude Code 等本地环境)
如果您使用的 AI Agent 支持挂载本地目录作为 Skill（如 ClaudeCode，可直接克隆本项目到： `~/.claude/skills`。
对话示例：
> 使用 hcf-skill：
> “户子，我月薪六千，攒了很久钱刚提了一辆比亚迪，你觉得怎么样？”

### 2. 自动安装
直接告诉AI：帮我安装 `https://github.com/Nahiyi/hcf-skill` 这个 skill。

## 警告与免责声明

1. **强攻击性预警**：本 Skill 的核心机制就是进行**无差别的人格贬低与消费歧视**。AI 的回复中将大量出现“你不配”、“安卓底层”、“割韭菜”等极化词汇。
2. **价值观隔离**：本 Skill 输出的所有观点（如极端拜金、社会达尔文主义、拉踩国货等）仅用于**人格模拟与社会学分析复盘**，绝对不代表开发者或 AI 本身的价值观。
3. **心理承受力**：请勿让心理承受能力较弱或容易产生容貌/阶层/收入焦虑的用户使用此 Skill。

## 贡献与扩展

如果您希望增强该 Skill 的“攻击力”或扩展其锐评领域，请直接向 `references/corpus.txt` 中追加新的原话语料，并在 `SKILL.md` 的交互示例中添加新的场景即可。
